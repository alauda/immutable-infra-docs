Use the following steps to install the component.

## Downloading

:::info
To use the immutable infrastructure, you need two cluster plugins from Customer Portal.
1. Download the app named **Alauda Container Platform Provider Kubeadm**.
2. Download the app named **Alauda Container Platform Provider DCS**.
