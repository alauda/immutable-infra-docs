# Upgrading Clusters on Bare Metal

This document provides instructions for upgrading Kubernetes clusters on bare metal servers.

## Status

📋 **Planned**

Documentation will be available when development begins.

## Key Considerations

Unlike VM-based platforms, bare metal upgrades use **in-place upgrade** to avoid lengthy physical machine reinstallation. Critical data persists on local disks during the upgrade process.
