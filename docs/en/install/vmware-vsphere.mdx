# VMware vSphere Provider Installation

This document describes how to install the VMware vSphere Infrastructure Provider for Immutable Infrastructure.

## Status

🚧 **In Development**

This provider is currently under active development. Documentation will be available when the provider is released.

## Planned Installation Steps

1. Download the vSphere Provider plugin from Customer Portal
2. Upload the plugin to <Term name="product" textCase="capitalize" />
3. Install the plugin on the `global` cluster
4. Configure vCenter credentials

For more information about the vSphere platform, see [Provider Overview](../overview/providers/vmware-vsphere.mdx).
