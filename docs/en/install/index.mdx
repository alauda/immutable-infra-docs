# Installation

<Overview />

Immutable Infrastructure requires installing infrastructure provider plugins to support different IaaS platforms. Each provider is an independent component with its own lifecycle and version.

For a list of supported providers, see [Infrastructure Providers](../overview/providers/index.mdx).

## Common Components

In addition to the infrastructure provider, the following components are required for all platforms:

- **Alauda Container Platform Kubeadm Provider**: Handles Kubernetes bootstrap and control plane management

## Installation Workflow

1. **Download Provider Plugin**: Obtain the provider plugin from Customer Portal
2. **Upload Plugin**: Upload the plugin to <Term name="product" textCase="capitalize" />
3. **Install Plugin**: Install the plugin on the `global` cluster
