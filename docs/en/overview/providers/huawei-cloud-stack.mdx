# Huawei Cloud Stack Provider

The Huawei Cloud Stack Infrastructure Provider enables Immutable Infrastructure on Huawei Cloud Stack platform.

## Overview

Huawei Cloud Stack is a hybrid cloud solution that extends public cloud capabilities to private data centers. The Cloud Stack Provider integrates with Huawei Cloud Stack to manage cloud resources for Kubernetes clusters.

This provider is currently under development. Documentation will be available when the provider is released.

## Planned Features

- Cloud resource management (instances, networks, storage)
- Integration with Huawei Cloud Stack APIs
- Support for hybrid cloud deployment scenarios
