# DCS Provider APIs

<Overview />

The Huawei DCS Infrastructure Provider defines custom resources for managing DCS infrastructure.

## Custom Resources

| Resource | Description | Documentation |
|----------|-------------|---------------|
| `DCSCluster` | Represents a Kubernetes cluster infrastructure on DCS | [DCSCluster](./dcscluster.mdx) |
| `DCSMachine` | Represents a virtual machine in DCS | [DCSMachine](./dcsmachine.mdx) |
| `DCSMachineTemplate` | Template for creating DCS machines | [DCSMachineTemplate](./dcsmachinetemplate.mdx) |
| `DCSIpHostnamePool` | Pool of IP addresses and hostnames | [DCSIpHostnamePool](./dcsiphostnamepool.mdx) |

## API Group

All DCS resources belong to the API group `infrastructure.cluster.x-k8s.io/v1beta1`.
