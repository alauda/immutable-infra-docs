# Immutable Infrastructure Documentation

Documentation site for Immutable Infrastructure, built with the [Doom](https://github.com/alauda/doom) framework.

## Requirements

- Node.js
- Yarn

## Quick Start

```bash
yarn install    # Install dependencies
yarn dev        # Start development server
yarn build      # Build production files
```

> **Note:** Restart `yarn dev` after modifying the sidebar configuration.

## Editor

We recommend using [Visual Studio Code](https://code.visualstudio.com/) with the [MDX extension](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx).
